# Installation
## Overview

The software packages are distributed as standalone executable applications and do not require installation of additional dependencies such as Python, MATLAB, or virtual environments.

All tools are designed to be easy to install and use, even for users without prior programming experience.

---

## System Requirements

- Operating Systems:Windows (64-bit)/Mac OS
- Hardware: Standard desktop or laptop computer
---

## Downloading the Software

Download the latest versions of the software packages from the following GitHub repositories:

- **Cone Detection ML**: [https://github.com/NIH-NEI/ConeDetectionML/tree/main](https://github.com/NIH-NEI/ConeDetectionML/tree/main)

- **Cone Segmentation ML**: [https://github.com/NIH-NEI/ConeSegmentationML/tree/main](https://github.com/NIH-NEI/ConeSegmentationML/tree/main)  

- **RPE Detection**: [https://github.com/NIH-NEI/RPE_Detection/tree/main](https://github.com/NIH-NEI/RPE_Detection/tree/main)  

Each repository contains platform-specific executable files.

---

## Installation Instructions
Details of the installation has been elaborately documented in the Readme files of the individual repositories. Below, we provide briefly provide steps to run the software with prebuilt executables. We recommend developers interested to run from source to visit the Github pages.

### Windows

1. Download the `.exe` installer or `.zip` package from the Release section of the GitHub repository.
2. If using the installer:
   - Double-click the `.exe` file.
   - Follow the installation wizard.
3. If using the `.zip` package:
   - Extract the contents to a local directory.
   - Open the application by running the executable file.

---

### Mac OS

1. Download the `.dmg` file from the Release section of the GitHub repository.
2. Open the `.dmg` file.
3. Drag the application to the Applications folder (if prompted).
4. Launch the application from the Applications folder.

---

## Running the Software

After installation:

1. Open the software application.
2. Load an AO image using the "Open" option. Example AO images provided in Example Images folder in Github.
3. Run detection or segmentation using the available tools.
4. Key features of the software packages are provided in the Cone Detection ML, Cone Segmentation ML and RPE Detection section.
